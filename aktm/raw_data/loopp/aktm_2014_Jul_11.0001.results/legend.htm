name - name of the matching structure

score - Summary score based on all the features listed in the table and computed using SVM.

homolog  - Number of homolog sequence (0 for the query sequence) scored highest for a  hit.

hits - Number of homolog sequences producing a given hit independently

length - Length of an alignment compared to the length of the query sequence

seq. ident. - Sequence  identity for an alignment

ENE all-atom - Energy calculated with an all-atom model produced from the best LOOPP alignment using MODELLER 6v2.

TE13 all-atom - Energy calculated using TE13 model with an all-atom model produced from the best LOOPP alignment using MODELLER 6v2.

zscore - Z-score for a given feature.

r-energy - Reverse energy: a difference between energy of a reversed chain (ordered from C- to N-terminus) and energy of a query chain

energy - Energy for a given alignment in a model ..

SEQ - Multi-feature alignment optimization and scoring

SEQ - Sequence-to-sequence alignment.

TRD - Sequence-to-structure (classical threading) alignment.

TRS  - Mixed model. Threading + sequence-to-sequence.

TSC - Secondary structure alignment.

TSS - Mixed model. Threading + secondary structure +sequence-to-sequencese.

PSM - Profile-based sequence alignment.

SRF - Exposed surface area-based scoring.

sec. str. pos. - Score for secondary structure alignment between query and target based on the positions of secondary structure elements.

sec. str. comp. - Score for secondary structure alignment between query and target based on the secondary structure types content.
